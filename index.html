El PROYECTO COLIBRI es una iniciativa ciudadana que pretende servir de puente entre la comunidad de programadores y periodistas y la web del Congreso de los Diputados de España para así facilitar el acceso a la información que en dicha web se puede encontrar.

REST API

Ofrecemos una API basada en servicios REST para consumición de datos del Proyecto. Puedes encontrar más información en la documentación oficial del mismo.

Visualización

Hemos desarrollado una aplicación web para consultar las votaciones del Congreso y queremos que sirva de base de futuros desarrollos que tanto nosotros como la comunidad pueda realizar.

100% libre

Tan libre como un pájaro. Ponemos a disposición de la comunidad tanto el código fuente del servidor de datos como los extractores de información de la web del Congreso junto con la aplicación web de visualización de datos.
