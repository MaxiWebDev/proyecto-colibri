La API ha sido desarrollada con django-tastypie y los datos son servidos en JSON. Si quieres probar las peticiones directamente desde el navegador es necesario que a√±adas "?format=json" a la url de la consulta.

La API esta formada por recursos; cada recurso es accesible a traves de la url http://proyectocolibri.es/api/v1/ seguido del nombre del recurso. Por ejemplo: http://proyectocolibri.es/api/v1/group/ , esto nos devolvera un JSON con todos los grupos parlamentarios y su informacion.

En los distintos recursos se han habilitado diferentes campos por los que poder filtrar. Para ello se le pasa como argumento el campo por el que se quiere filtrar y el valor que se desea. Por ejemplo, en el recurso de los Grupos parlamentarios podemos filtrar por el nombre de la siguiente manera:

http://proyectocolibri.es/api/v1/group/?format=json&name=G.P. LA IZQUIERDA PLURAL



Listado de Grupos parlmentarios

Este recurso nos devuelve los grupos parlamentarios que forman el congreso. En este recurso tambien obtenemos una relacion de los miembros que pertenecen a dicho partido.

Url Parametros
http://proyectocolibri.es/api/v1/group/ Campos para filtrar: id, name

Listado de Partidos

Con este recurso obtenemos un listado de los distintos partidos politicos del Congreso. En este recurso tambien obtenemos una relacion de los miembros que pertenecen a dicho partido.

Url Parametros
http://proyectocolibri.es/api/v1/party/ Campos para filtrar: id, name

Listado de Miembros

Con este recurso obtenemos todos los miembros del Congreso de los distintos grupos parlamentarios.

Url Parametros
http://proyectocolibri.es/api/v1/member/ Campos para filtrar: id, name

Listado de Votaciones

Recurso para obtener los votos de una determinada votacion en una sesion.

Url Parametros
http://proyectocolibri.es/api/v1/vote/ Campos para filtrar (obligatorios): session, number

Listado de Votaciones en cada Sesion

Recurso para obtener el listado de votaciones en cada sesion.

Url Parametros
http://proyectocolibri.es/api/v1/voting/ Campos para filtrar: session
