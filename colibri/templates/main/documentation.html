Documentacion

La API ha sido desarrollada con django-tastypie y los datos son servidos en JSON.

La API esta formada por recursos; cada recurso es accesible a traves de la url http://proyectocolibri.es/api/v1/ seguido del nombre del recurso. Por ejemplo: http://proyectocolibri.es/api/v1/group/ , esto nos devolvera un JSON con todos los grupos parlamentarios y su informacion.

En los distintos recursos se han habilitado diferentes campos por los que poder filtrar. Para ello se le pasa como argumento el campo por el que se quiere filtrar y el valor que se desea. Por ejemplo, en el recurso de los Grupos parlamentarios podemos filtrar por el nombre de la siguiente manera:

http://proyectocolibri.es/api/v1/group/?id=1



Listado de Grupos parlamentarios

Este recurso nos devuelve los grupos parlamentarios que forman el congreso. En este recurso tambien obtenemos una relacion de los miembros que pertenecen a dicho partido.

Url Parametros
http://proyectocolibri.es/api/v1/group/ Campos para filtrar: id, name
Detalle del recurso.

Listado de Partidos

Con este recurso obtenemos un listado de los distintos partidos politicos del Congreso.

Url Parametros
http://proyectocolibri.es/api/v1/party/ Campos para filtrar: id, name
Detalle del recurso.

Listado de Miembros

Con este recurso obtenemos todos los miembros del Congreso de los distintos grupos parlamentarios.

Url Parametros
http://proyectocolibri.es/api/v1/member/ Campos para filtrar: id, name, second_name
Detalle del recurso.

Listado de Miembros/partido

Con este recurso obtenemos todos los miembros del Congreso de los distintos grupos parlamentarios y el partido al que pertenecen.

Url Parametros
http://proyectocolibri.es/api/v1/groupmember/
Detalle del recurso.

Listado de Sesiones

Recurso para obtener las sesiones parlamentarias.

Url Parametros
http://proyectocolibri.es/api/v1/session/

Campos para filtrar: session, date

Campos para ordenar: date

Detalle del recurso.

Listado de Votoss

Recurso para obtener los votos de una determinada votacion en una sesion.

Url Parametros
http://proyectocolibri.es/api/v1/vote/ Campos para filtrar: session_number, number, member
Detalle del recurso.

Listado de Votos

Recurso para obtener todos los votos de todas las votaciones en todas las sesiones.

Url Parametros
http://proyectocolibri.es/api/v1/vote_full/
Detalle del recurso.

Listado de Votaciones

Recurso para obtener las votaciones de las sesiones.

Url Parametros
http://proyectocolibri.es/api/v1/voting/
Detalle del recurso.

Listado de Votaciones

Recurso para obtener las votaciones de las sesiones con los votos embebidos.

Url Parametros
http://proyectocolibri.es/api/v1/voting_full/
Detalle del recurso.
